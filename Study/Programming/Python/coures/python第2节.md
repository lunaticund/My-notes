### calculus：有值
由算子和算符组成
**除法：**
"//" ：整数除法，有一个是float，则结果是float
`7//-2`  : -4
`-7//2`  : -4
python按数学规则计算：下取整
"/"：一般除法
**取余：**
`5%-3` : 5//-3=-2 , 余数：5-(-3$\times$-2) = -1
`-5%3`=1 , `-5%-3`=-2

### statement：动作
**赋值：**
连续赋值：`a=b=c=5`
同步复制：`a,b=4,5`（没有时间先后）
不同于C，python可以利用同步赋值交换变量的值
```
a=4
b=6
a,b=b,a
print(a,b)
```
